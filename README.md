# 100Hires Portfolio Task — Step 1

This repository documents the first step of the 100Hires application process: setting up a development environment and publishing a GitHub repository.

## Tools Installed

- **Cursor IDE** — AI-powered code editor
- **Claude Code extension** — installed via Cursor's Extensions panel
- **Codex extension** — installed via Cursor's Extensions panel
- **Git** — for version control
- **GitHub account** — to host this repository

## Steps Completed

### 1. Install Cursor IDE

The first step is to download and install Cursor. Go to the official website: [https://cursor.com/download](https://cursor.com/download) and click **Download** for the desktop app. Once the installer is downloaded, run it to install the app.

After installation, open Cursor and sign up (or log in) with your email account. Cursor provides a 14-day free trial.

### 2. Install the Claude Code and Codex Extensions

To install extensions in Cursor:

- Press `CTRL + B` to open the sidebar/folder panel, then click the **Extensions** icon, **or**
- Press `CTRL + SHIFT + X` directly to open the Extensions search bar.

In the Extensions search bar:

1. Search for **Claude** and install the Claude Code extension.
2. Search for **Codex** and install the Codex extension.
3. Sign in to each extension when prompted.

> **Note:** You must be signed up or logged in to your Cursor account first. Without signing in to Cursor, you won't be able to install any extensions.

### 3. Create a Public GitHub Repository

Created a new public repository on GitHub to host this README.

### 4. Open the Repository in Cursor

Cloned the repository locally and opened it inside Cursor.

### 5. Create and Push This README

Created `README.md`, committed it, and pushed it to GitHub.

## Issues I Ran Into

- **Cursor login requirement for extensions** — At first I tried to install the Claude Code extension without signing in to Cursor and it wouldn't work. I learned that you need to sign up or log in to your Cursor account before any extension will install or activate.

- **Finding the Extensions panel** — I initially wasn't sure how to get to the Extensions view. I found two ways: `CTRL + B` to open the sidebar and then click the Extensions icon, or `CTRL + SHIFT + X` to jump straight to the Extensions search bar.

## What I Learned

- How to install and set up Cursor IDE and add AI coding extensions (Claude Code and Codex).
- How Cursor's extension system depends on being logged in to a Cursor account first.
